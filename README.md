# Data Structure Algorithm

## Fork it or Clone it

### Navigate to that directory
```
cd data-structure-algorithm
```

### Install Required dependencies
```
npm install
```

### Run test
```
npm test
```

### Run a sigle test file in watch mode:
```
npm test -- --watch test/stack.js
```
